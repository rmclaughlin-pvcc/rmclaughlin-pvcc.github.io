Operating Systems Terminology

Term Definition
Operating system the software that controls the overall operation of a computer
Job que jobs residing in mass storage waiting execution
FIFO First-in First-out
Interactive processing allows a program being executed to carry on dialog with a user through remote terminals
Time-sharing operating system that provides service to multiple users at thet same time
Multiprogramming time is divided into intervals and then the execution of each job is restricted to one interval at a time
Application software programs for preforming tasks particular to the machine's utilization
System software preforms tasks that are common to computer systems in general
Utility software preforming activity that is fundamental to computer installation but not included in the operating system
Graphical user interface (GUI) files and programs are represented pictorially on the display as icons
Shell communicated with users through textual messages using a keyboard and a monitor screen
Kernel internal part of an operating system
Device Driver software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine
File manager cooridinate the use of the machine's mass storage facilities
Memory manager cooridinates use of main memory
Virtual memory large "fictional" memory space created by paging
Boot strapping preformed by the computer when turned on (booting)
ROM read only memory
Deadlock condition where two or more programs are blocked from progressing because each is waiting on a resouce from one another
Spooling technique of holding data for output at a later but more convenient time
Super user (or administrator) establish accounts
Multi-factor authentication layered approach of securing data
