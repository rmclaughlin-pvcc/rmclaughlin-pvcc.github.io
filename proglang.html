Chapter 6 Terms: Programming Languages

Term Definition
Machine Language sequences of instructions that are encoded as numeric digits
Debugging locating and correcting errors
Assembler covert mnemonic expressions into machine language instructions
Assembly Language a mnemonic system for representing programs collectively
Translator program that translates expressed in high level primitives into machine language programs
Compiler what the translator programs were oftem called
Interpreter alternative to translators, used to implement third generation languages
Cross-platform software independent of an operating system's design and hardware design and is excutable through network
natural language ex. English, German, Latin
formal language programming languages
Machine Independence software not being reliant on the exact machine
High-Level Programming Language provide more abstractions for the programmer and make it easier for people to read and write program
Object-Oriented Programming (OOP) collection of units which is capable of preforming the actions that are immediately related to itself as well as requesting actions of other objects
Scripting Language subset of imperatve programming languages, used to perform administrative tasks rather than devolp complex program
Data Types
    integer: numeric data consisting of whole numbers
    float: numeric data that contains values rather than whole numbers
    string: sequence of characters or data
Primitive Data ex. int for interger and char for character
Data Structure conceptual shape or arrangement of data
Python Collection Data Structures
    list: Lists are used to store multiple items in a single variable
    tuple: Tuples are used to store multiple items in a single variable
    set: Sets are used to store multiple items in a single variable
    dictionary: Dictionaries are used to store data values in key:value pairs
Function set of instructions for preforming a task that can be used as an abstract tool by other program units
Local Variable limited to one part of the program
Global Variable variables scopes that are not limited to one part of the program
Source Program program in its original form
Object Program the translated version of a program
